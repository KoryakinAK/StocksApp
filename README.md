# StocksApp
Привет!
Буквально пара комментариев к проекту:
* Я создал второй таргет сборки, где вместо сетевых запросов подгружаются замоканные варианты ответов сервера без картинок. API часто падало последнюю неделю - может пригодится.
* У API не слишком комфортный лимит на количество запросов в минуту, поэтому приложение работает с небольшим захардкоженым списком акций вместо подгрузки условного S&P 500, но сам сетевой слой я постарался сделать модульным, чтобы было наглядно, что добавить запрос и прокинуть его результат в dataSource - вопрос считанных минут.
* Фильтр под поиском синхронизирован с переключателем в rightNavigationItem - это не баг, так и задумано.
