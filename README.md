# StocksApp
Пара замечаний к проекту:
* Есть второй таргет сборки, где вместо сетевых запросов подгружаются замоканные варианты ответов сервера без картинок. API иногда падает - может пригодится.
* Для Яндекс.ШМР - релиз 1.0 был сделан до дедлайна 5 апреля.

В приложении:
* Верстка только кодом
* Офлайн и онлайн загрузка данных: информация об акции принимается с разных эндпоинтов, синхронизируется через DispatchGroup. Либо из JSON в бандле приложения - в зависимости от собранного таргета.
* Немного неуместных анимаций: статичные UIView.animate, динамические UIViewPropertyAnimator, UIPercentDrivenInteractiveTransition. Реализация последней - по статье от команды ДоДо.
* Поиск с переключателем
* Почти Clean Swift (в процессе рефакторинга)
* Работа с избранным в UserDefaults
* Единственная зависимость - Kingfisher для иконок акций
